## [0.1.0] - 2026-01-22

### 🎉 Initial Release - Phase 0 Complete

This is the first release of dvlp-ksef - an open-source KSeF (Krajowy System e-Faktur) 
integration module with Microsoft Dataverse backend.

### ✨ Features

#### Web Application (Next.js 15)
- **Dashboard**: Overview of invoice statistics, KSeF connection status, quick actions
- **Invoice List**: Filterable table with incoming/outgoing tabs, search, status filters
- **Invoice Details**: Full invoice view with items table, VAT summary, XML preview
- **Sync Panel**: Session management, date range selection, import controls, operation log
- **Settings**: Company management, cost centers (MPK), security configuration

#### API (Azure Functions v4)
- **KSeF Session Management**: 
  - `POST /api/ksef/session` - Initialize interactive session
  - `GET /api/ksef/session` - Get active session status
  - `DELETE /api/ksef/session` - Terminate session
  
- **Invoice Operations**:
  - `POST /api/ksef/invoices/send` - Send single invoice
  - `POST /api/ksef/invoices/batch` - Batch send multiple invoices
  - `GET /api/ksef/invoices/{ref}` - Download invoice by KSeF reference
  - `GET /api/ksef/invoices/{ref}/status` - Check processing status
  - `POST /api/ksef/invoices/query` - Query invoices from KSeF
  - `POST /api/ksef/sync/incoming` - Sync incoming invoices
  - `GET /api/ksef/invoices/{ref}/upo` - Download UPO (official confirmation)

- **KSeF Client Library**:
  - Session lifecycle management with auto-refresh
  - Invoice XML builder (FA(2) schema)
  - Invoice XML parser
  - Multi-environment support (production, test, demo)

#### Deployment Infrastructure
- **Azure Bicep templates** for App Service, Functions, Key Vault
- **PowerShell scripts** for automated deployment
- **Dataverse managed solution** structure
