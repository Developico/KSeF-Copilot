## 🇵🇱 Polski

### Pierwsze publiczne wydanie KSeF Copilot

Pierwsze wydanie open-source rozwiązania do integracji z Krajowym Systemem e-Faktur (KSeF), zbudowanego w filozofii **API-First**.

### Kluczowe funkcjonalności

- **REST API** — Azure Functions v4 (Node.js/TypeScript) jako rdzeń produktu
- **Microsoft Dataverse** — integracja z platformą Power Platform jako backend
- **Synchronizacja faktur** — pobieranie faktur zakupowych z KSeF przez sesję
- **AI kategoryzacja** — automatyczna kategoryzacja faktur (MPK, kategoria, projekt) przez Azure OpenAI GPT-4o-mini
- **Wykrywanie anomalii** — silnik z 5 konfigurowalnymi regułami
- **Prognozowanie wydatków** — 5 algorytmów (wg kategorii, MPK, dostawcy)
- **Dashboard** — endpoint statystyk dla panelu analitycznego
- **Weryfikacja VAT** — sprawdzanie dostawców na białej liście VAT
- **Multi-walutowość** — PLN/EUR/USD z kursami NBP
- **Uwierzytelnianie** — Entra ID (Azure AD) OAuth2
- **Multi-tenant** — obsługa wielu firm (`settingId` / `tenantNip`)
- **Power Platform Custom Connector** — pełna definicja OpenAPI
- **Copilot Studio Agent** — agent dla Microsoft Teams z 14 narzędziami konektora
- **Natywna synchronizacja KSeF** — `POST /ksef/sync` przez sesję
- **Notatki do faktur** — dodawanie i pobieranie notatek wewnętrznych

## 🇬🇧 English

### First Public Release of KSeF Copilot

First open-source release of the Polish National e-Invoice System (KSeF) integration solution, built with an **API-First** philosophy.

### Key Features

- **REST API** — Azure Functions v4 (Node.js/TypeScript) as the core product
- **Microsoft Dataverse** — Power Platform backend integration
- **Invoice Synchronization** — fetching purchase invoices from KSeF via session
- **AI Categorization** — automatic invoice categorization (cost center, category, project) via Azure OpenAI GPT-4o-mini
- **Anomaly Detection** — engine with 5 configurable rules
- **Expense Forecasting** — 5 algorithms (by category, cost center, vendor)
- **Dashboard** — analytics statistics endpoint
- **VAT Verification** — vendor verification against the VAT Whitelist
- **Multi-currency** — PLN/EUR/USD with NBP exchange rates
- **Authentication** — Entra ID (Azure AD) OAuth2
- **Multi-tenant** — multiple company support (`settingId` / `tenantNip`)
- **Power Platform Custom Connector** — full OpenAPI definition
- **Copilot Studio Agent** — Microsoft Teams agent with 14 connector tools
- **Native KSeF Sync** — `POST /ksef/sync` via session
- **Invoice Notes** — internal notes for invoices
