Key Capabilities
🔄

Invoice Synchronization

Automatically sync purchase invoices from KSeF (Krajowy System e-Faktur) with configurable schedules and on-demand import.

🤖

AI Categorization

Intelligent invoice categorization powered by Azure OpenAI (GPT-4o) — automatic cost center, category and project assignment.

📊

Management Dashboard

Web-based dashboard for browsing, filtering and managing invoices. Track payment status, amounts and due dates at a glance.

🔒

KSeF authorization tokens stored in Azure Key Vault. RBAC with Administrator and Reader roles via Azure Entra ID.

📥

Export invoices to CSV/Excel. Webhook API for downstream integrations. E-mail notifications for new invoices and payment reminders.

About This Solution

KSeF Copilot is an open-source solution for integrating with the Krajowy System e-Faktur (Polish National e-Invoice System). It automates purchase invoice synchronization, provides AI-powered categorization via Azure OpenAI, and stores all data in Microsoft Dataverse. The backend runs on Azure Functions, while the dashboard is built with Next.js.

Architecture

  • Azure Functions — REST API (sync, import, categorize)
  • Next.js Dashboard — Invoice management UI
  • Microsoft Dataverse — Centralized data backend
  • Azure OpenAI (GPT-4o) — Intelligent categorization
  • Azure Key Vault — Secure token storage
Technology Stack

Getting Started

How to Get Started

  • Use the left navigation to access Invoices, Companies, Categories and Settings
  • Start by running a KSeF Sync to import your latest purchase invoices
  • Review imported invoices and apply AI Categorization for automatic cost center assignment
  • Track payment status (pending / paid) and set due date reminders
  • Export data to CSV or Excel for downstream accounting workflows
  • For support, contact our team at contact@developico.com
